🛒 Lista de Compras - Alimentos

Este é um projeto simples de lista de compras desenvolvido com HTML, CSS e JavaScript puro. Ele permite adicionar itens de forma rápida, marcar como concluídos e remover itens da lista.

🚀 Funcionalidades
Adicionar itens à lista de compras
Marcar itens como concluídos (riscar)
Remover itens individualmente
Interface simples e responsiva
Design limpo com foco em usabilidade
💻 Tecnologias utilizadas
HTML5
CSS3
JavaScript (Vanilla JS)
📌 Objetivo

O projeto foi criado para praticar manipulação do DOM com JavaScript, além de melhorar habilidades de front-end e lógica de programação.
